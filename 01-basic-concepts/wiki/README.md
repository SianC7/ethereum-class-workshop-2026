## Quick start

From the repository root:

```bash
cd 01-basic-concepts
yarn install
```

Run the staking tests. From `01-basic-concepts`:

```bash
yarn test test/StakingContract.ts
```

Note: the tests are expected to fail at first. If you complete the TODOs in the practical correctly, the tests should pass.

Or run all Hardhat tests from `01-basic-concepts`:

```bash
yarn test
```
