## Quick start

From the repository root:

```bash
cd 02-non-fungible-tokens
yarn install
```

Run the collectible tests. From `02-non-fungible-tokens`:

Note: the tests are expected to fail at first. If you complete the TODOs in the practical correctly, the tests should pass.

```bash
cd packages/hardhat
yarn test test/YourCollectible.ts
```

Or run all Hardhat tests from `02-non-fungible-tokens`:

```bash
yarn test
```
